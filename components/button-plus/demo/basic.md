## zh-CN

- 在antd.Button组件基础上封装, type属性值增加`del`, 删除按钮弹窗确认框
- 兼容antd.Button组件

## en-US

````jsx
import { message } from 'antd';
import { ButtonPlus } from 'wdd';

ReactDOM.render(
  <ButtonPlus type="del" onClick={() => message.success('删除成功')}>删除</ButtonPlus>,
  mountNode);
````
